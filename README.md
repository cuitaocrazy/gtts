# gtts

## install
```bash
npm i
npm install
```

## uninstall
npm uninstall -g

## example
```bash
gtts -i test.txt
# or
gtts Hello world!
```

## Usage

options:

	--tld -d: `com` `cn`. default: `cn`.
	--lang -l: `en` `zh`. default: `en`.
	--slow -s: `true` `false`. default: `false`.
	--input -i: input text file.
	--output -o: output mp3 file. default: `output.mp3`.
